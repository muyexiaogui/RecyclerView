2、项目说明
<1>、 该项目为Recycler使用，其中包括简单的Recycler，水平Recycler和瀑布流布局
<2>、该项目按照《第一行代码》Recycler讲解编写

3、路径说明
---activity  存放Activity代码
		HorizaonrecyclerActivity  水平Recycler的界面
		MainActivity  普通垂直Recycler的界面
		StaggerGridActivity  瀑布流界面
---adapter  存放适配器代码
		FruitAdapter HorizonAdapter StaggergridAdapter三个适配器内容一样，区别在于单条布局不一样，本人在第一遍对着书本敲的代码，为了加深影响，自己又写了两编
---entry 实体类

---layout 布局文件
        activity_horizaon_recycler.xml 水平Recycler布局的主界面
        activity_main.xml 垂直Recycler布局的主界面
        activity_stagger_grid.xml 瀑布流布局的主界面
        fruit_item.xml 垂直Recycler的单条布局界面
        fruit_item_horizon.xml 水平Recycler的单条布局界面
        fruit_item_stagger.xml  瀑布流单条布局界面
